Posisi: Pengembang Web Front-end & Back-end

Halo! Saya Vito, pencipta dari aplikasi "Film Favorit" ini. Aplikasi ini dirancang untuk membantu para pecinta film seperti Anda untuk dengan mudah mencari film terbaru atau klasik dari database OMDB yang luas dan menyimpan daftar film favorit Anda sendiri.

Visi saya adalah menciptakan pengalaman pencarian film yang intuitif, cepat, dan menyenangkan. Saya percaya bahwa setiap orang harus memiliki tempat pribadi untuk mengkurasi daftar film yang paling mereka hargai.

Aplikasi ini dibangun menggunakan teknologi modern seperti **Node.js (Express)** untuk backend, **MariaDB** sebagai database, dan **HTML, CSS (dengan Bootstrap),** serta **JavaScript murni** untuk frontend. Fitur-fitur seperti pencarian instan, manajemen favorit, responsivitas, dan mode gelap/terang ditambahkan untuk memastikan pengalaman pengguna yang optimal.

Terima kasih telah menggunakan Aplikasi Film Favorit! Jika Anda memiliki masukan atau pertanyaan, jangan ragu untuk menghubungi saya melalui media sosial di bawah ini.
